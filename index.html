📖 Về Game

Dodger Pro Max là một trò chơi nhập vai 2D hấp dẫn, nơi bạn trở thành một chim cánh cụt dũng cảm. Hãy thu thập tất cả những đồng xu quý giá, trả lời câu hỏi Python đầy thử thách, và chinh phục các tầng độ khó khác nhau. Bạn sẵn sàng trở thành tìm kiếm gần đúng tối cao?

✨ Tính Năng Chính

🪙Thu Thập Xu

Thu thập tất cả những đồng xu trên bản đồ và mở cổng thoát để chiến thắng.

🐍Câu Hỏi Python

Trả lời câu hỏi về lập trình Python để tồn tại nhân đôi khi gặp quái vật.

⚡Quyền Năng

Khô Shield, Speed, Jump, và Slow - mỗi quyền năng có tác dụng riêng!

😈Quái Vật AI

Gặp phải những kẻ thù thông minh với hành động truy đuổi động.

🎯4 Độ Khó

Easy, Normal, Hard, Insane - thách thức bản thân ở mọi mức độ.

🗺️Đa Bản Đồ

Khám phá nhiều bản đồ với thiết kế độc đáo và những vật cản có vị.

🎮 Cách Chơi

Điều Khiển

← → Chuyển động trái/phải
SPACE Nhảy (bấm 2 lần để nhảy kép)
ESC Tạm dừng game
ENTER Khi gặp quái vật: xác nhận câu trả lời

Mục Tiêu

1️⃣ Thu thập tất cả các đồng xu trên bản đồ
2️⃣ Tránh hoặc đánh bại quái vật bằng cách trả lời câu hỏi Python
3️⃣ Đến cổng thoát khi đã thu thập hết xu
4️⃣ Chiến thắng và đạt điểm cao!

🌟 Hệ Thống Quyền Năng

🛡️ Shield

Bảo vệ bản thân khỏi quái vật một lần. Khi được bảo vệ, bạn không thất bại!

⚡ Speed

Tăng tốc độ chuyển động gấp đôi để trốn tránh hoặc nhanh chóng thu thập xu.

🔼 Jump

Tăng độ cao nhảy để tới những vị trí khó tiếp cận hoặc vượt qua chướng ngại vật.

🐢 Slow

Làm chậm tốc độ của quái vật để có thêm thời gian thoát thoát.

🎯 Độ Khó

  • Easy 🟢 - Quái vật chuyển động chậm, tốc độ 3 px/frame. Hoàn hảo để bắt đầu!
  • Normal 🟡 - Cân bằng về thách thức. Quái vật cao chiều tăng từng bước.
  • Hard 🔴 - Quái vật nhanh (tốc độ 7 px/frame). Cần kỹ năng tốt!
  • Insane 💜 - Mode cực khó với quái vật siêu nhanh (tốc độ 9 px/frame). Chỉ dành cho cao thủ!

🛠️ Công Nghệ

Python 3.12 - Ngôn ngữ lập trình chính
Pygame 2.6.1 - Thư viện game 2D
Modular Design - Hệ thống module riêng biệt (Characters, Items, Enemies)
Optimized Rendering - Hiệu suất cao với caching và batch rendering

💻 Cách Chơi & Cài Đặt

🌐 Cách 1: Chơi Trực Tiếp Trên Web (Dễ Nhất!)

Bạn không cần cài đặt gì cả! Chỉ cần click nút "CHƠI NGAY TRÊN WEB" ở trên để chơi game thẳng trên trình duyệt bằng Replit. Không cần cài Python, Pygame hay bất cứ gì khác!

🚀 Mở Trên Replit Ngay

📥 Cách 2: Cài Đặt Trên Máy Tính

Nếu bạn muốn chơi offline hoặc có máy tính riêng:

# 1. Clone repository
git clone https://github.com/YourUsername/Dodger-main.git
cd Dodger-main

# 2. Cài đặt Pygame
pip install pygame

# 3. Chạy game
python dodger.py

⚙️ Setup Replit (Nếu Tự Tạo)

  1. 1. Truy cập https://replit.com
  2. 2. Click "Import from GitHub" → Dán link repo: https://github.com/YourUsername/Dodger-main
  3. 3. Chọn Python, Replit tự cài Pygame
  4. 4. Click "Run" → Game sẽ chạy ngay lập tức!
  5. 5. Chia sẻ link Replit với bạn bè để họ chơi online
